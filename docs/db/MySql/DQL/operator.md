### 运算符

| 算数运算符 | 说明 |
| ---------- | ---- |
| +          | 加号 |
| -          | 减号 |
| *          | 乘号 |
| /          | 除号 |
| %          | 求余 |

sql语言支持位运算(如>>,<<,&,|,~,^);

| 比较运算符 | 说明                                                        |
| ---------- | ----------------------------------------------------------- |
| =          | 相等                                                        |
| <=>        | 安全等于（等于号跟赋值语句一样我是真的没见过）js甚至有===号 |
| !=         | 不等于                                                      |
| <>         | 不等于                                                      |
| >          | 大于                                                        |
| <          | 小于                                                        |
| >=         | 大于等于                                                    |
| <=         | 小于等于                                                    |
| !>         | 不大于                                                      |
| !<         | 不小于                                                      |
| LIKE       | 表示操作数与模式匹配                                        |
| IS NULL    | 判断是否是NULL                                              |
| LEAST      | 存在俩个以上多个参数时返回最小值                            |
| GREATEST   | 存在俩个以上多个参数时返回最大值                            |
| REGEXP     | 正则表达式匹配                                              |
